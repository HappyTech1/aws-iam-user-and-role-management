# AWS IAM User and Role Management Project

## Project Overview

This project demonstrates the implementation of Identity and Access Management (IAM) in Amazon Web Services (AWS). The project focuses on creating IAM users, groups, roles, and policies while following security best practices and the principle of least privilege.

## Technologies Used

* Amazon Web Services (AWS)
* Identity and Access Management (IAM)
* AWS Console
* Security Policies
* Role-Based Access Control (RBAC)

---

## Project Objectives

* Create IAM users
* Create IAM groups
* Assign permissions using policies
* Create IAM roles
* Implement least-privilege access
* Improve AWS account security
